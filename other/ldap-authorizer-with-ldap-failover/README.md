# How to ensure high availability of LDAP using DNS SRV Records

## Objective

Quickly test [DNS SRV Records for LDAP](https://ldap.com/dns-srv-records-for-ldap/) in order to ensure high availability of LDAP.

## How to run

Simply run:

```
$ ./start-ldap.sh
```

or with LDAPS:

```
$ ./start-ldaps.sh
```
